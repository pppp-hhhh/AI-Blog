# 欢迎来到我的Nuxt.js博客

很高兴你能来到我的博客！这是一个使用[Nuxt.js](https://nuxt.com/)构建的现代化博客网站，具有服务端渲染(SSR)、静态生成(SSG)等强大功能。

## 博客特色

### 🚀 现代化技术栈
- **Nuxt.js 3** - 基于Vue.js的全栈框架
- **Tailwind CSS** - 实用优先的CSS框架
- **@nuxt/content** - 强大的内容管理模块
- **Pinia** - Vue.js状态管理

### 📱 响应式设计
博客采用移动优先的设计理念，在各种设备上都能提供良好的阅读体验。

### 🔍 SEO优化
- 服务端渲染(SSR)支持
- 自动生成元标签
- 结构化数据标记
- 友好的URL结构

### ✍️ 写作体验
使用Markdown格式写作，支持代码高亮、数学公式、图表等丰富内容。

## 功能介绍

### 文章管理
- 分类和标签系统
- 全文搜索功能
- 阅读时间统计
- 相关文章推荐

### 内容展示
- 优雅的文章排版
- 代码语法高亮
- 图片懒加载
- 响应式表格

### 用户交互
- 文章导航
- 分页浏览
- 标签云展示
- 分类筛选

## 开始使用

你可以通过以下方式开始探索：

1. **浏览文章** - 查看所有技术文章
2. **按分类查看** - 根据兴趣选择分类
3. **搜索内容** - 使用搜索功能找到特定文章
4. **阅读推荐** - 查看相关文章推荐

## 技术细节

博客使用Nuxt.js的约定式路由，自动生成页面路由：

```javascript
// pages/articles/index.vue - 文章列表页面
// pages/articles/[...slug].vue - 文章详情页面
// pages/categories/index.vue - 分类页面
// pages/about.vue - 关于页面
```

内容使用Markdown格式存储，通过@nuxt/content模块解析和渲染：

```markdown
---
title: "文章标题"
description: "文章描述"
date: "2024-01-15"
category: "frontend"
tags: ["Vue.js", "Nuxt.js"]
---

# 文章内容

正文内容使用Markdown格式编写。
```
